Alors c'est quoi un système de gestion de base de donnée (SGBD)?

Un système de gestion de base de données (abr. SGBD) est un logiciel système servant à stocker, à manipuler ou gérer, et à partager des données dans une base de données, en garantissant la qualité, la pérennité et la confidentialité des informations, tout en cachant la complexité des opérations.

Qui utilise les SGBD?

Parmi les utilisateurs des SGBD, nous citons les administrateurs de base de donnée , les concepteurs de base donnée, les développeurs ainsi que tout autres utilisateurs pour qui l'utilisation ou la manipulation de donnée s'avère un chemin inévitable. Pour précision, les SGBD sont incontournables dans le domaine des big data, des data sciences, de l'intelligence artificielle. Elles sont utilisées dans diverses domaines notamment l'économie, la finance, le domaine bancaire, la psychologie, la medecine. Elles contribuent à resumer et à faciliter l'exploitation d'informations provenant de diverses sources. Le système de gestion de base donnée, une arme vitale pour le développement des entreprises

les differents types de système de gestion de base de donnée

on distingue plusieurs types de système de gestion de base de donnée. Nous citons entre autre les bases de données relationelles, les bases de données hiérachiques, les bases de données orientées réseau, les bases de données orientées objet enfin les bases de données orientées document.

• les bases de données relationelles

Pour ce type de système de gestion de base de donnée, la structuration des données est effectuée dans des lignes de tableaux. L’avantage est de pouvoir établir des relations entre les différentes lignes et de pouvoir les présenter dans des colonnes.

• les bases de données hiérachiques

Pour ce type de système de gestion de base de donnée, les différentes données sont reliées dans une arborescence et suivent un système hiérarchique.

• les bases de données orientées reseau

Pour ce type de système de gestion de base de donnée, les données sont structurées dans un réseau

• les bases de données orientées objet

Ce type de système de gestion de base de donnée, en plus d'intégrer les relations entre les ensembles de données, les héritages jouent également un rôle essentiel dans ces bases de données. Les objets peuvent ainsi transmettre des propriétés à d’autres objets selon un schéma défini par le SGBD.

• les bases de données orientées document

ce type de système de gestion de base de donnée, prévois un stockage des ensembles de données dans différents documents.
